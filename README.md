## CalorieTracker
___
Приложение для фиксирования и ослеживания калорий пользователя с учетом потребляемой им еды.

Приложение написано с использованием SpringBoot 3.2.0, Java 17
Приложение настроено для использования PostgreSQL, для подключения используй `user: root`, `password: password`,
либо измени на свои в `application.properties`

___
Установка: `mvn clean install`

___
Запуск: `java -jar /{path to file}/CalorieTracker.jar`
___
После запуска документация длступна по адресу: http://localhost:8080/swagger-ui/index.html
